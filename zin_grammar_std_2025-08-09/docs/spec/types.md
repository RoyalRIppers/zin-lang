# Types de base (v0.1)

- `nombre` / `num` : 64-bit flottant signé (provisoire) — objectif: `i64`/`f64` distincts plus tard.
- `texte` / `str`  : chaîne UTF-8 immuable.
- `booléen`/`bool` : bool (true/false).

> À venir: `entier`/`int`, `octet`/`byte`, `tableau`/`array[T]`, `dictionnaire`/`map[K,V]`, `structure`/`struct`.
